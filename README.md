# student-social-media-ml
Goal: Build a model that predicts whether a student reports high, medium, or low relationship satisfaction based on their social media usage.

Key Steps in the Notebook

Visualize country distribution, usage patterns, sleep, mental health
Create Addiction_Level (High/Low)
Train/test split
Build ML pipeline (scaling + encoding + RandomForest)
Evaluate model (classification report + confusion matrix)
Feature importance
KMeans clustering
PCA visualization of personas
Add persona names
Generate personalized recommendations
Export final dataset
